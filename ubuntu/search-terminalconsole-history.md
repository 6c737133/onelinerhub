# Search terminal/console history

```bash
history | grep "git"
```

- `history` - prints entire commands history for current user
- `grep` - will filter only specified values
- `"git"` - sample filter to use

group: history

## Example: 
```bash
history | grep "git"
```
```
 1992  git push
 1993  git status
 2002  history | grep "git"
```
