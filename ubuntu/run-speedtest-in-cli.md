# Run speedtest in CLI

```bash
apt install speedtest-cli
speedtest-cli
```

- `apt install` - installs specified package on Ubuntu
- `speedtest-cli` - users speedtest.net service to measure internet speed on the machine

## Example: 
```bash
speedtest-cli
```
