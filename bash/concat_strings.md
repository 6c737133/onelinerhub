# How to concatenate strings in bash

```bash
a="hi"; b="!"; echo "$a$b"
```

- `a="hi"` - first variable ```a``` with ```hi``` text
- `b="!"` - second variable ```b``` with ```!``` text
- `"$a$b"` - concatenation of two variables ```a``` and ```b```, will print ```hi!```
