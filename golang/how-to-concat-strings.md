# How to concat strings

```go
result := "hi" + " all!"
```

- `result` - final variable with concatenated strings value
- ` + ` - concatenate two strings

## Example: 
```go
package main
  
import "fmt"
  
func main() {
  var str1 string
  str1 = "Hi"
  
  result := str1 + " all!"
  fmt.Println(result)
}
```
```
Hi all!

```
