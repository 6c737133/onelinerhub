# List all indexes

### Execute this Manticosearch query using [Mysql protocol](/manticoresearch/mysql-client-connection-example):

```sql
SHOW TABLES
```

- `SHOW TABLES` - Will list all indexes

## Example: 
```sql
SHOW TABLES
```
```
+--------+------+
| Index  | Type |
+--------+------+
| index1 | rt   |
+--------+------+
1 row in set (0,00 sec)
```
