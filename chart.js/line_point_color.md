# Set line chart point color

```javascript
new Chart('chart', {
  // ...
  data: {
    datasets: [{ backgroundColor: 'rgba(10, 194, 130, 0.5)' }]
  }
});
```

- 'chart' - id of ```canvas``` element
- backgroundColor: 'rgba(10, 194, 130, 0.5)' - set point color to semi-transparent green

group: line_style
