# Set line chart line color

```javascript
new Chart('chart', {
  // ...
  data: {
    datasets: [{ data: [1, 2, 3], borderColor: '#0ac282' }]
  }
});
```

- 'chart' - id of ```canvas``` element
- data: \[1, 2, 3\] - example dataset
- borderColor: '#0ac282' - change line color to green

group: line_style
