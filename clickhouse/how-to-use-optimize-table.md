# How to use optimize table

### **Warning.** This query can take a lot of time and resources for large tables.

```sql
OPTIMIZE TABLE tbl
```

- `OPTIMIZE TABLE` - will optimize table data for the selected table
- `tbl` - name of the table to optimize

## Example: 
```sql
OPTIMIZE TABLE tbl
```
