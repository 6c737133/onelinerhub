# Convert text to hex

```sql
SELECT hex('example')
```

- `hex(` - converts given string to HEX number
- `example` - string to convert

group: hex

## Example: 
```sql
SELECT hex('example')
```
```
┌─hex('example')─┐
│ 6578616D706C65 │
└────────────────┘

```
