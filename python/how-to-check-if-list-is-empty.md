# How to check if list is empty

```python
if list:
  print("list is not empty")
else:
  print("list is empty")
```


## Example: 
```python
list1 = [1, 2, 3]
list2 = []

if list1:
  print("list1 is not empty")
else:
  print("list1 is empty")
  
if list2:
  print("list2 is not empty")
else:
  print("list2 is empty")
```
```
list1 is not empty
list2 is empty

```
