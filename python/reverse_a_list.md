# Reverse a list using a function 

```python
list_a = ["foo", "bar", "baz"]
list_a.reverse()
```

- .reverse() - function takes no input and does not require any imports

# Example

```python
list_a = ["foo", "bar", "baz"]
list_a.reverse()
```
```bash
['baz', 'bar', 'foo']
```
