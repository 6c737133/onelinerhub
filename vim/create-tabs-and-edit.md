# How to create multiple VIM tabs

Open a file in VIM
```console
vim file.txt
```

On Command Mode, type 
```console
:tabedit file2.txt
```
```console
:tabedit file3.txt
```

To switch between tabs, type this command to go to the NEXT tab on the editor(file2)
```console
:tabn
```

And this to go to the PREVIOUS tab
```console
:tabp
```
