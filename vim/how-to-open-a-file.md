# How to open new file from vim

### Press `Esc` to [switch to command mode](/vim/how-to-switch-to-command-mode) and then type (then press `Enter`):

```text
:e <filename>
```

- ``vim`` - use vim
- ``:`` - tell vim we want to use a command
- ```e``` - key for "edit"
- ``<filename>`` - the name of the file
